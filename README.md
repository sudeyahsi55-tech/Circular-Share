# CircularShareApp ♻️📦

**CircularShareApp**, kullanıcıların kullanmadıkları veya az kullandıkları eşyaları çevrelerindeki insanlarla paylaşmalarını, ödünç vermelerini veya takas etmelerini sağlayan bir döngüsel ekonomi ve eşya paylaşım platformudur. 

Bu proje, tüketim çılgınlığını azaltmak, topluluk içi dayanışmayı artırmak ve sürdürülebilir bir geleceğe katkıda bulunmak amacıyla bir dönem ödevi olarak geliştirilmiştir.

---

## 🚀 Özellikler

*   **Kullanıcı Giriş/Kayıt Sistemi:** Güvenli profil oluşturma ve giriş işlemleri.
*   **Eşya İlanı Oluşturma:** Kullanıcıların paylaşmak istedikleri eşyaların fotoğraflarını, durumlarını ve açıklamalarını ekleyebileceği ilan paneli.
*   **Kategori ve Filtreleme:** İhtiyaç duyulan eşyalara (Kitap, Elektronik, Kamp Malzemeleri, Giysi vb.) hızlıca ulaşma imkanı.
*   **Ödünç Alma/Talep Etme:** Beğenilen bir eşya için ilan sahibine pratik bir şekilde talep gönderebilme.
*   **Harita/Konum Entegrasyonu (Varsa):** Yakındaki paylaşımları görebilme özelliği.

## 📦 Kurulum ve Çalıştırma

Projeyi yerel bilgisayarınızda çalıştırmak için aşağıdaki adımları takip edebilirsiniz:

1.  **Projeyi Klonlayın:**
```bash
    git clone [https://github.com/KULLANICI_ADIN/CircularShareApp.git](https://github.com/KULLANICI_ADIN/CircularShareApp.git)
    ```
2.  **Proje Klasörüne Gidin:**
```bash
    cd CircularShareApp
    ```
3.  **Bağımlılıkları Yükleyin:**
```bash
    # Örn: Node.js projesi ise
    npm install
    # Örn: Flutter projesi ise
    flutter pub get
    ```
4.  **Projeyi Başlatın:**
```bash
    # Örn: Node.js için
    npm start
    # Örn: Flutter için
    flutter run
    ```
